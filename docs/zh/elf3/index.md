# ELF3 精灵3 通用人形机器人

精灵3 是一款身高1.45米的中型人形机器人，其运动参数和同等身高的人类接近。

---

## 产品特征

### 机械结构
- 先进执行器：全身31个中空行星关节电机，支持机器人做出奔跑、跳跃、空翻、跑酷等高动态运动。    
- 骨架式设计：1、极致减轻重量；2、用户可以通过加装外壳改变机器人外形、加装更多设备。    
- 高可靠性：中空行星电机大幅减少精灵3的线束；抗摔倒设计匹配机器人用户的可靠性需求。    

### 电子和通信架构
- 高速PECI-CANFD 通信，整机控制频率 >1000Hz    
- 无EtherCat协议支持    
- CPU: NUC Intel i7 gen 13 +  GPU：选装适配，支持运动控制+视觉端到端任务控制；

### 传感器
- IMU、深度相机、激光雷达、RGB相机等多模态传感器接入；

### 电源管理
- 支持电池热插拔，确保机器人持续工作能力；    

### 开发者支持
- 基于CANFD的电机通信协议，开发者可以直接控制机器人的电机；    
- 基于ROS2的控制框架SDK，开发者可以一键部署自己的控制策略网络，到仿真环境（Mujoco based）和真机硬件；
