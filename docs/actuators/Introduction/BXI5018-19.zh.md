## 关节电机参数表

| 参数项 | 规格值 | 单位 |
| :--- | :--- | :--- |
| **额定电压** | 24–48 | V |
| **空载转速** | 200 | RPM |
| **输出端额定转速** | 100 | RPM |
| **额定扭矩** | 11 | Nm |
| **峰值扭矩** | 35 | Nm |
| **峰值相电流** | 30 | A(rms) |
| **减速比** | 19.5 | — |
| **重量** | 0.55 | kg |

> **注**：以上参数均为理论值，实际值可能因工况有所偏差。
